How does javascript works?

Answer : javascript is a client site, high level, scripting language. javascript allows us to make web pages interactive. With javascript we can change anything of html and css. javascript is a interpreted language and this is not a compiled language that's why browers interpreter read the javascript code and show us the output. In javascript there has a execution context and everything happens inside that execution context. javascript maintain a call stack and it works to main the order of execution context. Call stack is similiar to stack that means it also works with LAST IN FIRST Out.
